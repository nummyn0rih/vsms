VSMS · инструмент снабженца. Вид «Доска» (B5 v2). Карточка стоит в колонке по дню прибытия. Одно-фермерская карточка — основной примитив (один фермер, его культуры дня чипами). Сборка: режим выбора (кнопка ☑ в шапке дня) → отметить ≥2 одно-фермерских карточки → панель «Собрать в машину». Машина — агрегат фермеров с действием «Разобрать». Перетаскивание плановой карточки и собранной машины задаёт прибытие = день колонки, отправление = −2 рабочих дня. sent-машина таскается, но меняется только дата прибытия (отправление зафиксировано). arrived / accepted — залочены (без хвата, замок). Неделя влезает по ширине (6 рабочих дней, колонки 1fr); сайдбар сворачивается. Даты/сервер в прототипе не реализованы.
