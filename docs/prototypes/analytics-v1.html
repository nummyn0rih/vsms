Дашборд сезона

Сводный read-only экран по текущему сезону — заменяет плейсхолдер /analytics. KPI-полоса + сетка из 4 графиков; агрегаты считаются на лету из отгрузок / приёмок / контрактов (не хранятся). Год-к-году в этот срез не входит — истории пока нет. Токены наследуются из системы: hairline #ebebeb, Geist + Geist Mono tabular-nums; серии культур — из Culture.color, нейтральные метрики (недели, ТК) — графитовая шкала, брак — нейтральный янтарь. Числа иллюстративные; в проде графики — Recharts, селектор сезона — URL ?season=.

Решения / вопросы

Открыто на ревью раскладки v1 — до кода на Recharts.

Линия динамики — суммарная или по культурам?
вопросv1 — одна суммарная линия/area (принято по всем культурам). Разрез по культурам (несколько линий, серии из Culture.color) — вероятный v2: при 4+ культурах график читается хуже, нужен тумблер «сумма / по культурам». Утвердить: достаточно ли суммарной для первого релиза?
Выполнение по культурам — горизонтальные бары
склоняемсяГоризонтальные: длинные названия культур читаются в строку, % и «принято / план» встают справа без поворота подписей. Вертикальные оставлены для метрик с короткими подписями (брак, рейсы).
Брак — разрез по фермерам, а не только культурам?
вопросv1 — брак по культурам (доля брака от принятого). Разрез «брак по фермерам» полезен для претензий, но это отдельный таб/дровилл — не на обзорный дашборд. Нужен ли drill-down «культура → фермеры» в v1 или в карточку фермера (v2)?
«Осталось ~N машин» — как считать средний вес рейса?
решеноv1 — грубая оценка: недобор / плановый вес рейса из норм (≈3,6 т). Фактический средний вес рейса как отдельная метрика — не в этом срезе (уводит от плана). Подпись явно помечает оценку («≈»).
Рейсы ТК — сгруппированные бары или две серии стопкой?
склоняемсяСгруппированные (овощные + материальные рядом): раздельный счёт по BR-14 читается сразу, стопка бы прятала соотношение. Графит вместо статусных цветов — ТК не про статус отгрузки.

VSMS · инструмент снабженца. Статичный макет дашборда /analytics (v1) — раскладка, оси и цвета до кода на Recharts. Read-only, все роли (аналитика — R). Год-к-году, печатные формы и фактический средний вес рейса — вне этого среза. Серии культур — Culture.color; недели и ТК — графитовая шкала; брак — нейтральный янтарь, не алярм-красный.
