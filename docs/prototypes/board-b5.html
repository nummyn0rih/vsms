VSMS · инструмент снабженца. Статичный макет вида «Доска» (B5). Карточка стоит в колонке по дню прибытия. Перетаскивание плановой карточки между колонками задаёт прибытие = день колонки, отправление = −2 рабочих дня (даты/сервер в прототипе не реализованы). Карточки sent+ (Отправлена / Прибыла / Принята) залочены — без хвата, с замком. Полоса прогресса считается от суммы по культурам видимой недели; перелёт цели отмечен акцентной риской. Цвета — статусам (status-fills) и культурам (Culture.color); хром нейтрален.
