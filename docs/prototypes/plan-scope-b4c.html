План — состав культур недели

Combobox «Культуры · N» в тулбаре вида «План» управляет составом недели (WeeklyPlanScope) — какие культуры показываются и планируются на этой неделе. Цель — убрать пустые строки: из 10–15 культур обычно задействованы 2–5. Это общий server-state (редактирует admin, видят все), а не личный фильтр ленты. Визуально наследует combobox фильтра «Сырьё», но управляет только пустыми культурами: культуру с целью или отгрузками снять нельзя (BR-23) — она закреплена и поясняется причиной.

Состав · рабочее состояние
01
Combobox открыт — закреплённые, выбранные и свободные
Кнопка «Культуры · 5» с нейтральной заливкой и счётчиком (как активный фильтр ленты). В списке: 3 закреплённых пункта (есть отгрузки / есть цель — отмечены и неотключаемы), 2 выбранных вручную и 8 свободных. Матрица позади показывает только 5 строк состава — ради этой компактности всё и затевается. Свободные чекбоксы: вкл = строка появляется пустой, готовой под ввод; выкл = строка исчезает. Клик по закреплённому поясняет причину, а не молча игнорируется.

Combobox · анатомия
02
Три типа пунктов в списке
Поиск сверху, чекбоксы культур (чип цвета + название), нейтральная заливка. Закреплённые пункты — отмечены и неотключаемы (серый чекбокс + причина), свободные — обычные ink-чекбоксы. Подвал: «сохраняется сразу» + счётчик. Закрытие — клик вне / Esc.
A
Поиск фильтрует список по названию — для длинных перечней (15+ культур).
B
Закреплена · «есть отгрузки». На неделе есть неудалённые отгрузки культуры → видна по BR-23 независимо от состава. Чекбокс disabled, снять нельзя.
C
Закреплена · «есть цель». Задана WeeklyPlan на неделю → тоже неотключаема. Причина названа явно, клик не игнорируется молча.
D
Свободная, выбрана. Пустая культура в составе: строка в матрице есть, ждёт ввода цели. Снятие убирает строку.
E
Свободная, не выбрана. Включение добавит культуру в состав (WeeklyPlanScope) и пустую строку под план.

VSMS · инструмент снабженца. Статичный макет управления составом культур недели (B4c). Состав — WeeklyPlanScope (общий server-state, редактирует admin): combobox управляет только пустыми культурами. По BR-23 строка видна, если культура в составе ИЛИ есть цель ИЛИ есть отгрузки — поэтому культуру с активностью снять нельзя, она закреплена и поясняется причиной. Combobox не меняет расчёты прогресса и подытогов (BR-22): после выбора в матрице остаются только видимые строки, «Итого» и headline считаются по видимому/плановому набору как раньше. Токены, бары и матрица наследуются из plan-view-b4; статусные цвета отгрузок в план-домене не используются.
