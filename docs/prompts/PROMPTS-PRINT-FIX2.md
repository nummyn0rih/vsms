# PROMPTS — print-fix2: кнопка «Печать» на Сводке/Плане (реальная причина) + проверка центрирования

> Самодостаточный фикс-промпт (запускать в НОВОМ чате Claude Code без прочего контекста).
> Симптом: на видах «Сводка» (/shipments) и «План» (/planner) НЕТ кнопки «Печать», хотя код её передаёт.
> Причина найдена: `components/shell/FeedToolbar.tsx` рендерит `printSlot` ВНУТРИ блока `{showFilters && (...)}`
