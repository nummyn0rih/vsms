# PROMPTS — print-4: «Развернуть все недели» + печать ленты в 3 режимах

> Доработка ленты и её печатного листа. Сейчас печатается ТОЛЬКО выбранная неделя. Нужно: кнопка
> **«Развернуть/Свернуть все недели»** в ленте и **выбор режима печати** — текущая неделя · раскрытые недели ·
> вся лента по фильтрам (многостраничный лист). **Без миграции, read-only.** Каркас печати (print-1) и лист
> «Отгрузки» уже есть — расширяем их.

## Решения (зафиксированы с пользователем)
| Развилка | Выбор |
|---|---|
| Режимы печати | **Три**: `week` — текущая неделя (как сейчас, дефолт) · `expanded` — только раскрытые на экране недели · `all` — вся лента (все недели, прошедшие фильтры). |
| Фильтры | Во всех режимах применяются текущие фильтры ленты (уже сериализуются в URL печати). |
| Раскрытие | Кнопка-тумблер «Развернуть все / Свернуть все» в тулбаре ленты (управляет существующим состоянием сворачивания недель). |
| Многостраничность | Недели идут секциями подряд (шапка недели + дни + подытог недели), без принудительного разрыва на каждую неделю; не рвать машину/день по возможности. В конце — общий итог по всем включённым неделям. |
| Семантика (важно) | В режиме `expanded` свёрнутая неделя в документ НЕ попадает — это осознанный выбор пользователя. В шапке листа явно писать режим и состав. |
