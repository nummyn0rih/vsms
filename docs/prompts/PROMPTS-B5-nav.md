# PROMPTS — B5-nav: реструктуризация навигации + глобальная неделя в URL

> Разнести виды по маршрутам, перевести неделю в URL (глобально, дефолт — текущая), переименовать ярлыки,
> попутно фикс расхождения инпутов в «Плане». Без миграции.

## Целевая структура
- **/shipments** (Лента отгрузок): виды «Лента» (бывш. «Таблица») + «Сводка» (бывш. «Heatmap»).
- **/planner** (Планировщик, заглушку заменить): виды «План» (перенести с /shipments) + «Доска» (перенести).
- **Неделя — общая для всего приложения через URL** `?week=YYYY-Www`, дефолт = текущая ISO-неделя
  (`currentSeasonWeek`). Переключение вида (`?view=`) и переход между /shipments↔/planner — неделю сохраняют.
