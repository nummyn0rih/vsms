# PROMPTS — print-3: печать Аналитика (с графиками)

> Последний срез печатных форм. Лист **Аналитика сезона** — **A4 portrait, С ГРАФИКАМИ** (KPI + 4 графика,
> в отличие от рабочих видов). На каркасе print-1 (`PrintSheet`/print-CSS/bare-роуты). Источник —
> `getSeasonAnalytics`; графики — переиспользовать компоненты дашборда. **Без миграции, read-only, без денег.**
> Прототип: `print-views-v1.html` (лист 05).

## Решения (зафиксированы)
| Развилка | Выбор |
|---|---|
| Графика | Аналитика — ЕДИНСТВЕННЫЙ печатный вид с графикой (наглядность). KPI-полоса + 4 графика: выполнение по культурам (CSS-бары), динамика приёмки (Area), % брака (Bar), рейсы ТК (grouped Bar). Серии в цвете. |
| Ориентация | **A4 portrait** (по прототипу: KPI сверху + сетка графиков 2×2 под лист). |
| Данные | `getSeasonAnalytics` (тот же, что дашборд `/analytics`). Графики — переиспользовать компоненты дашборда (Recharts) в print-layout. Не дублировать агрегацию. |
| Деньги | НЕ выводить (натуральные единицы: т/%/рейсы). |
| Контекст | Сезон из URL `?season`. Кнопка «Печать» на /analytics. |
