# PROMPTS — shipment-revert: откат статуса продуктовой отгрузки + правка до акта

> Закрыть пробел: машина в «Прибыла» неоткатываема и неисправима. Дать админу цепочку откатов
> accepted→arrived→sent→planned и правку позиций в статусе planned. Без миграции.

## Цепочка (правка = через возврат в план)
| Откат | Действие | Гард |
|---|---|---|
| accepted→arrived | revertAct (есть) | — |
| **arrived→sent (НОВЫЙ)** | нетто-сторно плеча прибытия (-1↔завод) + очистить actual_weight_kg + status=sent | нет привязанного AcceptanceAct |
| sent→planned | revertShipmentToPlanned (есть) | status=sent |
| правка позиций | существующая форма в статусе planned | — |

Прямую правку в sent/arrived НЕ даём: тара двинута по плановому весу при отправке → правка десинхронит плечи.
Безопасный путь: откатить в план → править → отправить заново (цепочка откатов разматывает плечи нетто-сторно).
