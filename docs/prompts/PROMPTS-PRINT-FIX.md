# PROMPTS — print-fix: экранная раскладка print-роутов + кнопки Сводка/План + тултип

> Самодостаточный фикс-промпт (можно запускать в НОВОМ чате Claude Code без прочего контекста).
> Печатные формы (print-1/2/3) реализованы: bare print-роуты `app/print/{shipments,acceptance,summary,plan,analytics}`.
> Три правки: (A) экранная подача листа сломана (прижат влево, обрезан, нет скролла); (B) на Сводке и Плане нет
> кнопки «Печать»; (C) страховка от Recharts-тултипа на печати. **Без миграции, read-only. Вывод НА ПЕЧАТИ
> (`@media print`, чистый A4) НЕ менять — только экранная подача и добавление кнопок.**

3. Кнопки «Печать» на видах «Сводка» (/shipments) и «План» (/planner) → /print/summary|plan с контекстом недели.
