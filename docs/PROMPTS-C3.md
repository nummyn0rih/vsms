# PROMPTS — Этап C, срез C3 (деньги)

Замыкаем контур поставки: принятый вес → стоимость (принятый × цена строки) → выполнение строк
контракта → зона 3 «Принято». Дробим на 4 проверяемые единицы (правило: 1 промпт = 1 единица;
новый экран → сперва дизайн):

- **C3a** — расчёт стоимости + выполнения строк (backend, чистые функции). БЕЗ UI. ← этот файл, промпт 1.
- **C3b** — дизайн зоны 3 «Принято» (Claude Design). ← этот файл, промпт 2.
- **C3c** — зона 3 «Принято» код (просмотр актов + откат admin). Детализируем после C3a + утверждения C3b.
- **C3d** — выполнение/стоимость в экране «Контракты». Детализируем после C3a.

Порядок: **C3a (деньги-расчёт, самостоятелен) → C3b (дизайн) параллельно → C3c → C3d.**
